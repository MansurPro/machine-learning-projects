# Chapter 03: Linear Regression

This chapter contains code and experiments related to linear regression, including:
- Implementation of linear regression algorithms
- Visualization of results
- Analysis of model performance


## Files
- `Linear_regression.ipynb`: Jupyter notebook with code and explanations for linear regression.
- `House_price_predictions.ipynb`: Notebook for predicting house prices using linear regression techniques.
- `Hyderabad.csv`: Dataset used for house price prediction examples.
